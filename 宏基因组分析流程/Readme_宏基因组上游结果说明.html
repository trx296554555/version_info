结果文件说明

  • 物种分类结果: all_bracken_res.tsv

    1. 文件格式：第一列为NCBI TaxID，第二列Taxonomy分类，第3-最后列为样本ID，矩阵数据为count数;
    2. 文件里Taxon列的每个__前的字母表示分类水平: (U)nclassified<未分类>, (R)oot<根/总read数>, (D)omain<域>, (K)ingdom<界>, (P)hylum<门>, (C)lass<纲>, (O)rder<目>, (F)amily<科>, (G)enus<属>, or (S)pecies<种>;
    3. 文件里面的数据值是比对到相应物种的原始Reads数【raw count】;
    4. 注意：除了alpha多样性，其他基于物种分类的宏基因组下游分析推荐将原始数据使用【相对丰度Rle】转换为【相对数据】或使用【CLR】转换为【成分数据】进行后续分析 REF。

  • 功能注释结果：reads_func_anno

    1. 文件格式：与物种分类结果类似，首列为功能条目的ID，后续列为样本ID;
    2. 额外提供了注释ID的详细描述信息文件在【06_annotation_info】文件夹中，用于ID与Name的对应，也可在各自数据库官网自行查询;
    3. COG的注释结果来着EGGNOG DATABASE 详细注释信息可以在NCBI官网或者EGGNOG官网查询
    功能类型 结果文件 数据库官网 额外描述信息文件 数据格式
    EGGNOG all.cog.tsv EGGNOG官网 Normalized Counts
    COG分类 all.cog_cate.tsv COG官网 Normalized Counts
    预测基因 all.pref.tsv NCBI官网 Normalized Counts
    KEGG all.ko.tsv KEGG官网 Normalized Counts
    GO all.go.tsv GO官网 Normalized Counts
    耐药基因 all.rgi.tsv CARD官网 RGI描述信息 TPM
    碳水化合物酶 all.dbcan.tsv CAZY CAZy描述信息 TPM
    毒力因子 all.vfdb.tsv VFDB官网 vfdb描述信息 TPM
